SwiftzJS - Edusupplicant 的 NodeJS 实现
==========

**Swiftz** 项目基于对 Edusupplicant 3.7.8 的分析，致力于构建 Edusupplicant 的全平台、开源、便携的替代品。

**本项目仅供学习研究之用，严禁用于攻击、入侵及其它任何非法用途。**


## 状态

目前处于频繁开发中。


## 安装

```
$ sudo npm install -g swiftz
```


## 使用

```
$ sw login 账号
```
